# Банковско поврзување / Banking (PSD2)
## For: both

### Преглед
Поврзете ја вашата банковна сметка со Facturino преку PSD2 (Open Banking) за автоматски увоз на трансакции и усогласување со фактури. Достапно од Standard пакетот.

### Како да поврзете банковна сметка
1. Одете на **Банка** во левото мени (или **Поставки** → **Банковски конекции**)
2. Кликнете **Поврзи банка**
3. Изберете ја вашата **банка** од листата на поддржани банки
4. Ќе бидете пренасочени на **банковната страница** за автентикација
5. Најавете се со вашите банковни креденцијали
6. Одобрете го **пристапот** до сметката (read-only — само преглед)
7. По успешно поврзување, ќе бидете вратени во Facturino

### Поддржани банки
- Сите банки во Северна Македонија кои поддржуваат PSD2
- Поврзувањето е безбедно — Facturino само ги чита трансакциите, не може да прави плаќања

### Увоз на трансакции
- По поврзување, трансакциите автоматски се увезуваат
- Видете ги во **Банка** → **Трансакции**
- Секоја трансакција покажува: датум, опис, износ, салдо

### Автоматско усогласување (Auto-Reconciliation)
- Системот автоматски ги усогласува банковните трансакции со вашите фактури
- Кога препознае уплата за фактура, автоматски ја означува како платена
- Можете рачно да ги усогласите неусогласените трансакции

### Правила за усогласување (Matching Rules)
1. Одете на **Банка** → **Правила**
2. Кликнете **Ново правило**
3. Дефинирајте услов: ако описот содржи „КИРИЈА", тогаш...
4. Дефинирајте акција: креирај трошок во категорија „Кирија"
5. Системот автоматски ги применува правилата на нови трансакции

### Историја на увоз
- Видете ги сите претходни увози во **Банка** → **Историја**
- Секој увоз покажува: датум, број на трансакции, статус

### Рачно усогласување
1. Одете на **Банка** → **Трансакции**
2. Најдете ја неусогласената трансакција
3. Кликнете **Усогласи**
4. Изберете: поврзи со фактура, креирај плаќање, или креирај трошок
5. Потврдете

### Обнова на поврзување
- PSD2 поврзувањата истекуваат на секои 90 дена
- Системот ќе ве извести кога треба повторно да се поврзете
- Процесот е ист — кликнете **Обнови** и најавете се повторно на банката

### Лимити
- Достапно од **Standard** пакет и погоре
- API лимит: 15 барања/минута (автоматски управувано)

### Поврзани функции
- Плаќања (автоматско евидентирање)
- Фактури (автоматско означување како платени)
- Трошоци (автоматско креирање од трансакции)
- Правила за усогласување (автоматизација)
