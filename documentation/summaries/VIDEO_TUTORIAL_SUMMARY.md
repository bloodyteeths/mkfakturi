## üìù NOTES FOR PRODUCTION TEAM

### Key Considerations
1. **Cyrillic Text:** Ensure UTF-8 encoding throughout production
2. **Pacing:** Slower than normal - allow time to read on-screen text
3. **Cursor Highlighting:** Essential for clarity in complex screens
4. **Example Data:** Use realistic Macedonian business names
5. **Validation Errors:** Show real errors to prepare users

### Common Pitfalls to Avoid
- ‚ùå Moving cursor too fast
- ‚ùå Not pausing after important information
- ‚ùå Background noise in audio
- ‚ùå Garbled Cyrillic characters
- ‚ùå Skipping error scenarios
- ‚ùå Too much text on screen at once
- ‚ùå Inconsistent visual style

### Best Practices
- ‚úì Record in segments, not all at once
- ‚úì Use consistent browser window size
- ‚úì Highlight important UI elements clearly
- ‚úì Show both success and error scenarios
- ‚úì Maintain conversational, friendly tone
- ‚úì Test on actual Macedonian data
- ‚úì Get feedback from non-technical users
