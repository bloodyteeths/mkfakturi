THE STRATEGY

PHASE 1: Trojan Horse (2026)

Accounting app = Distribution

SMEs adopt Facturino for invoicing. When they connect their bank, we become the data layer.

This is exactly how Plaid started — inside Venmo.

PHASE 2: Platform (2027)

Open the API

Thousands connected → expose API to developers. Lending apps, expense tools, analytics — all build on us.

PHASE 3: Regional (2027-28)

Replicate across Balkans

PHASE 4: Payments (2028+)

From data to money movement
