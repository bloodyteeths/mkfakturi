# Проекти / Projects
## For: both

### Преглед
Проектите ви овозможуваат да ги групирате фактурите, трошоците и плаќањата по проект за подобро следење на профитабилноста.

### Како да креирате проект
1. Одете на **Проекти** во левото мени
2. Кликнете **Нов проект**
3. Внесете:
   - **Име** на проектот
   - **Код** (кратка ознака)
   - **Клиент** (опционално)
   - **Буџет** и валута
   - **Датум на почеток** и **датум на крај**
4. Кликнете **Зачувај**

### Статуси на проект
| Статус | Опис |
|--------|------|
| Open | Нов проект, уште не е почнат |
| In Progress | Активен проект |
| Completed | Завршен проект |
| On Hold | Паузиран проект |
| Cancelled | Откажан проект |

### Поврзување со документи
- Кога креирате фактура, трошок или плаќање, можете да го доделите на проект
- Сите документи поврзани со проектот се видливи во проектниот преглед

### Преглед на проект
- Кликнете на проект за да видите:
  - Вкупни приходи (фактури)
  - Вкупни трошоци
  - Плаќања примени
  - Нето профитабилност
- Филтрирајте по датум за конкретен период

### Достапност
- Достапна од **Starter** пакетот
