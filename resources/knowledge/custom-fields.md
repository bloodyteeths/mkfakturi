# Прилагодени полиња / Custom Fields
## For: both

### Преглед
Додадете сопствени полиња на фактури, понуди, клиенти, артикли и кредитни ноти за да зачувате дополнителни информации специфични за вашиот бизнис.

### Како да додадете прилагодено поле
1. Одете на **Поставки** → **Прилагодени полиња**
2. Кликнете **Додади поле**
3. Поставете:
   - **Име** на полето
   - **Тип**: текст, број, датум, паѓачко мени (dropdown), чекбокс
   - **Каде се применува**: фактури, понуди, клиенти, артикли, кредитни ноти
   - **Задолжително**: да/не
4. Кликнете **Зачувај**

### Типови на полиња
| Тип | Опис |
|-----|------|
| Text | Текстуално поле |
| Number | Нумеричко поле |
| Date | Поле за датум |
| Dropdown | Паѓачко мени со предефинирани опции |
| Checkbox | Поле за означување (да/не) |

### Каде се прикажуваат
- Полињата се прикажуваат во формуларите за креирање и уредување
- Пример: ако додадете поле "Број на проект" на фактури, ќе се појави при креирање на секоја фактура

### Ограничувања по пакет
- Бројот на прилагодени полиња зависи од вашиот претплатнички пакет
