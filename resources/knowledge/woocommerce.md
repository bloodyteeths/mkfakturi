# WooCommerce интеграција / WooCommerce Integration
## For: both

### Преглед
Facturino може да се поврзе со вашата WooCommerce онлајн продавница за автоматски увоз на нарачки и креирање на фактури. Оваа функција е достапна за Business пакетот (€59/месец).

### Како да поврзете WooCommerce
1. Одете на **Поставки** → **Интеграции** → **WooCommerce**
2. Внесете ги податоците за вашата продавница:
   - **Store URL**: URL на вашата WooCommerce продавница (пр. https://myshop.mk)
   - **Consumer Key**: API клуч од WooCommerce (генерирајте го во WC Admin → Settings → REST API)
   - **Consumer Secret**: Тајниот клуч од WooCommerce
3. Кликнете **Тестирај конекција** за да потврдите
4. Кликнете **Зачувај**

### Генерирање на WooCommerce API клучеви
1. Во вашиот WooCommerce админ панел, одете на **Settings** → **Advanced** → **REST API**
2. Кликнете **Add Key**
3. Дадете опис (пр. "Facturino Integration")
4. Изберете **Read** дозволи
5. Кликнете **Generate API key**
6. Копирајте ги Consumer Key и Consumer Secret

### Автоматска синхронизација
- Вклучете **Auto-sync** во поставките
- Изберете фреквенција: на 15 минути, 1 час, или 4 часа
- Системот автоматски ги повлекува нови нарачки

### Рачна синхронизација
- Кликнете **Синхронизирај сега** за моментален увоз на нарачки

### Кои податоци се синхронизираат
Од WooCommerce кон Facturino:
- **Клиент**: име на компанија, адреса, е-пошта, телефон
- **Нарачки**: линиски артикли (име, количина, цена, ДДВ)
- **Износи**: поднаслов, данок, попуст, достава, вкупно
- **Метаподатоци**: број на нарачка, статус, начин на плаќање, валута

### Статуси на нарачки кои се синхронизираат
- Processing (во обработка)
- Completed (завршени)
- On-hold (на чекање)

### ДДВ маппинг
- Поставете маппинг меѓу WooCommerce даночни класи и Facturino ДДВ стапки
- Пример: WC "Standard" → Facturino ДДВ 18%
- Пример: WC "Reduced" → Facturino ДДВ 5%

### Историја на синхронизација
- Видете ги сите претходни синхронизации во табелата на дното
- Секоја синхронизација покажува: време, број на синхронизирани нарачки, статус

### Потребен пакет
- WooCommerce интеграцијата е достапна од **Business** пакетот (€59/месец)
