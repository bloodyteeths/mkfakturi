# Рокови / Deadlines
## For: both

### Преглед
Следете ги важните рокови — даночни обврски, плаќања, поднесување на извештаи и други деловни рокови.

### Како да додадете рок
1. Одете на **Рокови** во менито
2. Кликнете **Нов рок**
3. Внесете:
   - **Наслов** (пр. "ДДВ пријава за јануари")
   - **Тип** (даночен, плаќање, извештај, друго)
   - **Датум на достасување**
   - **Опис** (опционално)
4. Кликнете **Зачувај**

### Статуси
| Статус | Опис |
|--------|------|
| Upcoming (Претстоен) | Рокот е во иднина |
| Due Today (Денес) | Рокот е денес |
| Overdue (Задоцнет) | Рокот поминал |
| Completed (Завршен) | Обврската е исполнета |

### Преглед на рокови
- Видете ги сите рокови подредени по датум
- Филтрирајте по тип и статус
- Задоцнетите рокови се означени со црвена боја

### За сметководители (партнери)
- Партнерите имаат преглед на рокови за сите нивни клиенти
- Можат да управуваат со рокови за секоја компанија одделно
