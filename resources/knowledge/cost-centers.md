# Трошковни центри / Cost Centers
## For: both

### Преглед
Трошковните центри овозможуваат алоцирање на приходи и расходи по оддели, проекти или организациски единици. Ова дава јасна слика за профитабилноста на секој сегмент од бизнисот. Патека: `/admin/cost-centers`.

### Како да креирате трошковен центар
1. Одете на **Трошковни центри** во левото мени
2. Кликнете **Нов трошковен центар**
3. Внесете:
   - **Име** (пр. „Продажба", „Маркетинг", „Проект Алфа")
   - **Код** (опционално — интерна шифра, пр. „TC-001")
   - **Опис** (опционално)
4. Кликнете **Зачувај**

### Користење во документи
Трошковните центри можат да се доделат на:

- **Фактури**: При креирање фактура, изберете трошковен центар
- **Сметки (Bills)**: При евидентирање на набавки
- **Трошоци (Expenses)**: При евидентирање на директни трошоци
- **Книжења (Journal Entries)**: При рачно книжење

За секој документ:
1. Отворете го формуларот за креирање/уредување
2. Најдете го полето **Трошковен центар**
3. Изберете од листата на достапни центри
4. Зачувајте

### Извештаи по трошковен центар
1. Одете на **Извештаи** или **Буџети**
2. Филтрирајте по **трошковен центар**
3. Видете ги приходите и расходите за конкретниот оддел/проект
4. Споредете ги перформансите меѓу различни центри

### Буџетирање по трошковен центар
- При креирање на буџет, можете да го поврзете со трошковен центар
- „Буџет vs. Реалност" автоматски ги филтрира само трансакциите за тој центар
- Ова е клучно за одделенско буџетирање

### Примери за примена
- **По оддел**: Продажба, Маркетинг, Администрација, IT, Логистика
- **По проект**: Проект А, Проект Б — корисно за проектно сметководство
- **По локација**: Канцеларија Скопје, Канцеларија Битола
- **По линија на производи**: Услуги, Софтвер, Хардвер

### Поврзани функции
- Буџети (планирање по трошковен центар)
- Фактури (алоцирање на приходи)
- Трошоци (алоцирање на расходи)
- Книжења (детално рачно алоцирање)
- Извештаи (филтрирање по центар)
