# Competitor Gap Analysis: Facturino vs Pantheon vs Zonel

> Last updated: 2026-03-04

## Competitors

- **Pantheon** (DataLab, Slovenia) — Desktop ERP, ~4,000 MK companies, ~€70/mo accountant license
- **Zonel HELIX** (Zonel Software, Skopje) — Desktop accounting, ~1,500 MK companies, unpublished pricing

---

### Priority Summary

**Should build next:**
1. Interest calculations — commonly needed, low effort
2. Workflow/approval chains — accounting bureaus need this for document verification

**Can wait:**
3. Travel expenses — niche, few users need it
4. Everything else — outside core scope or very low demand
