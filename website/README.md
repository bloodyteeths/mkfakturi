Facturino marketing website

Tech
- Next.js 15 (App Router), React 19
- Tailwind CSS 4
- Locale routes with middleware redirect to default `mk`: `/mk`, `/sq`, `/tr`

Pages per locale
- `/[locale]/` Landing (hero, why‑different, benefits, how‑it‑works, CTA)
- `/[locale]/features`
- `/[locale]/how-it-works`
- `/[locale]/e-faktura`
- `/[locale]/pricing`
- `/[locale]/security`
- `/[locale]/contact`

Dev
- Requires Node.js >= 20.9 for Next.js 16.x in this template.
- Install: `npm install`
- Run dev: `npm run dev` then open http://localhost:3000 (auto‑redirects to `/mk`).
- Build (Railway): `npm run build`
- Production start: `npm start`
